## 安装

先安装nvm（node版本管理工具），然后使用nvm来安装npm。

参考：[安装npm](#docs/imooc_vue_basic#环境安装)。


## 查看帮助

### npm

npm帮助入口。


### npm help <term>

#### 示例：查看 npm install 命令具体有哪些参数

```
npm help install
```
