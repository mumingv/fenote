# Sublime

## 快捷键

|类别        |功能        |快捷键           |备注               |
|------------|------------|-----------------|-------------------|
|            |多行游标    |`<Ctrl+D>`       |                   |
|复制内容    |复制当前行并将光标定</br>位在下一行的相同位置|`<Ctrl+Shift+D>`|选择多行可实现多</br>行内容复制|
|删除内容    |删除当前行  |`<Ctrl+Shift+K>` |选择多行可实现多</br>行内容删除|
|移动内容    |将当前行移动到上一行|`<Ctrl+Shift+UP>`|选择多行可实现多</br>行内容移动|
|移动内容    |将当前行移动到下一行|`<Ctrl+Shift+DOWN>`|选择多行可实现多</br>行内容移动|
