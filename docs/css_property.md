# CSS属性汇总

## 字体

######   

|属性名称           	|属性含义           	|备注               |
|-------------------|-------------------|-------------------|
|font-family        |字体样式             |取值示例：sans-serif|


## 文本

######  

|属性名称           	|属性含义           	|备注               		|
|-------------------|-------------------|-----------------------|
|color              |文本字体颜色       	|不会改变链接文本的颜色		|
|background-color   |背景颜色           	||
|text-decoration	|文本装饰				|如：下划线为underline		|


## 边框

######  

|属性名称           	|属性含义           	|备注               |
|-------------------|-------------------|-------------------|
|border-bottom      |下边框           	|属性值顺序：with style color|
