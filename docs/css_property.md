# CSS属性汇总

## 文本

######  

|属性名称          	|属性含义           |备注               		|
|-------------------|-------------------|---------------------------|
|font-family        |字体系列           |取值示例：sans-serif|
|font-size          |字体大小           ||
|color              |文本颜色           |不会改变链接文本的颜色		|
|font-weight        |字体粗细           ||
|text-decoration	|文本装饰			|如：下划线为underline		|


## 边框

######  

|属性名称           	|属性含义           	|备注               |
|-------------------|-------------------|-------------------|
|border-bottom      |下边框           	|属性值顺序：with style color|
|background-color   |背景颜色           ||
